## Name

Vidminno

## Description

This project is a frontend for Learning Management System Vidminno.

### Prerequisites

`npm` must be installed. Then you must install dependencies using `npm install`

### Run

To run this project locally using `npm`:

`npm run start`

#### Stack:

1. React
2. Redux
3. RTK Query
4. react-router
5. rsuite
6. react-markdown
7. react-chartjs-2
